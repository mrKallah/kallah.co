Reinforcement learning for Driving using A3C

In this group project, the aim was to automate the driving for a remote-controlled vehicle with a front mounted camera. This was done using the A3C algorithm in PyTorch. The idea was to train the model in a simulated environment using TORCS and then apply the training from the TORCS to the real-world model. My contribution here was primarily in image pre-processing, helping out with the TORCS simulator and in tying the different parts of the project together. My main focus was on creating a pre-processing step that would consistently take any image from the simulator or the real world and make them as similar as possible. This was done by removing as much unimportant information as possible. An example video of the preprocessing steps is attached below and the source code and my co-authors can be found at GitHub

Digital Analysis of Paintings using TensorFlow

This project focused on investigating the ability to use a Convolutional Neural Network (CNN) to classify which artist created a painting using raw TensorFlow. The dataset used in this project was gathered by contacting various museums and asking for permission. With the data gathered, the network was trained using those paintings alongside some photos taken of random objects and people. When the network had finished training the model was saved and based on the parameters used, it could be light weight enough to run on a simple web server. (Link to the web classifier) (please note the images uploaded are available to the public, and will be saved until the next image is uploaded) (If broken, this is likely due to php being outdated, feel free to contact me using my email below) The code for this project is also available on GitHub.

After the project was finished, the ability to do multi class predictions was implemented. This came at the cost of a slight decrease in accuracy. A large update to the pre-processing steps was also added, improving both usability and performance. Lastly, the save and load method was changed to automatically save and load the used parameters in order to avoid manually changing parameters when loading a model.
